# 千川监控助手

一款 Chrome 浏览器扩展，用于实时监控抖音电商罗盘千川投放数据，并支持通过企业微信机器人推送数据报告与预警通知。

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 📊 **实时数据抓取** | 自动抓取千川消耗、成交金额、GPM、在线人数、ROI 等核心指标 |
| 📡 **定时推送** | 按设定间隔（默认 5 分钟）推送数据报告到企业微信 |
| ⚠️ **多维度预警** | 消耗阈值预警、在线人数预警、低在线人数预警、ROI 异常预警 |
| 📺 **直播状态检测** | 智能识别直播状态，未开播时自动暂停推送 |
| 🔧 **XPath 自定义** | 支持自定义 XPath 规则，适配罗盘页面布局变化 |
| 🔄 **自动刷新** | 页面隐藏超过设定时间后自动刷新，保持数据最新 |
| 📱 **侧边栏模式** | Chrome 侧边栏展示，悬浮按钮一键展开/收起 |
| 📈 **数据历史** | 保留最近 100 条历史记录，支持查询 |

## 安装方式

### 方式一：开发者模式加载（推荐）

1. 下载本项目代码到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择项目文件夹即可

## 使用说明

### 1. 配置企业微信 Webhook

1. 在企业微信群中添加机器人，获取 Webhook 地址
2. 打开插件侧边栏（点击扩展图标或页面悬浮按钮）
3. 将 Webhook 地址填入配置项并保存

### 2. 数据监控

- 打开 [抖音电商罗盘](https://compass.jinritemai.com/) 千川页面
- 插件会自动抓取页面数据
- 数据将按配置间隔自动推送到企业微信

### 3. 预警设置

| 预警类型 | 触发条件 |
|----------|----------|
| 消耗阈值 | 每分钟消耗超过设定金额 |
| 在线人数 | 在线人数超过设定阈值 |
| 低在线人数 | 在线人数低于设定值并持续一段时间 |
| ROI 异常 | ROI 低于设定阈值 |

## 数据抓取说明

插件通过以下方式获取页面数据：

1. **XPath 定位**（主要方式）
   - 默认配置了罗盘页面各指标的 XPath
   - 可在侧边栏自定义修改

2. **标签匹配**（备用方式）
   - XPath 失效时自动尝试通过类名和标签匹配
   - 支持 Odometer 数字组件解析

3. **数据校验**
   - 自动过滤科学计数法等异常值
   - 金额上限校验（1 亿）
   - 退款金额合理性检查

## 注意事项

1. 本插件仅用于个人数据监控，请勿用于非法用途
2. 需要保持罗盘页面打开才能正常抓取数据
3. 如页面布局更新导致 XPath 失效，请手动更新配置
4. 建议开启直播状态检测，避免未开播时无效推送
